# Deployment Guide for Your Portfolio

Your portfolio has been successfully built and is ready for deployment. The built files are located in the `dist` folder.

## Deployment Options

### 1. Netlify (Recommended - Free)
1. Sign up at [netlify.com](https://netlify.com)
2. Drag and drop the `dist` folder to the deployment area
3. Your site will be live immediately with a Netlify URL
4. You can customize the domain in the settings

### 2. Vercel (Alternative - Free)
1. Sign up at [vercel.com](https://vercel.com)
2. Create a new project
3. Import your portfolio repository or upload the `dist` folder
4. Deploy and enjoy your custom Vercel URL

### 3. GitHub Pages
1. Create a GitHub repository for your portfolio
2. Push your code to the repository
3. Enable GitHub Pages in the repository settings
4. Select the `gh-pages` branch as source

### 4. Manual Deployment
1. Upload the contents of the `dist` folder to any web server
2. Ensure the server is configured to serve static files
3. Your portfolio will be accessible via the server's URL
